Aprende en comunidad

Codificación Activa es una comunidad formada por programadores de diferentes países de habla hispana en la que se comparten conocimientos, se realizan proyectos colaborativos y lo más importante, ayudar a las personas a adentrarse al mundo de la programación.

Que vas a encontrar en codificacion activa?

Acá encontraras diferentes tipos de recursos, proyectos, ideas y personas. ¡No te faltarán razones para quedarte!

Proyectos

Los proyectos serán propuestos por los administradores o por los demás miembros de la comunidad, tú puedes proponer el tuyo, sea en cualquier lenguaje a cualquier nivel, así como podrás participar voluntariamente en otros de tu gusto.

Comunidad

Los miembros de la comunidad son quienes mantienen la plataforma en pie, sin ellos no se podriá lograr lo que hemos logrado y seguiremos haciendo, siempre bajo supervisión de los administradores. Cualquier comportamiento indebido podría alertar a los administradores, así que sigue los Terminos y Condiciones.

Recursos

Como ya se ha mencionado, en nuestra comunidad encontraras diferentes recursos agregados por nuestros propios miembros, seguro encontrarás el que necesitas en nuestra amplia colección de libros en pdf, videos, enlaces y más.

¿Que vas a aprender?

JavaScript

Aprende Javascript y Node.js

Script del lado del usuario + framework del lado del servidor.

Aprende a usar el principal script del lado del usuario para páginas web, y además tienes la opción de aprender a usar una de sus framework del lado del servidor: Node.js.

Haz tu web dinámica con JavaScript y gestiona tu servidor gracias a Node.js.

Ruby

Aprende Ruby y Ruby On Rails

Scripts de escritorio + framework para back-end de páginas web.

Aprende un excelente script para desarrollar aplicaciones de escritorio, sin duda una buena opción para realizar cálculos y otras cosas, además de una framework para el back-end de páginas web: Ruby On Rails

Excelente para cálculos complejos de manera fácil y rápida.

PHP

Aprende PHP

Preprocesador para HTML y script del lado del servidor.

Aprende PHP, un clásico y poderoso script del lado del servidor con todo tipo de funcionalidades, como correos electrónicos, bases de datos, y muchas cosas más.

Soportado por muchos hosting comerciales y utilizado por grandes plataformas modernas. ¡Incluso Facebook lo utiliza para sus servidores!
